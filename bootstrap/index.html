Project description

Ansible+Terraform

Google Cloud Platform

Я с помощью Terraform поднял сервер на платформе Google Cloud Platform. Затем я с помощью Ansible я развернул на сервере Docker контейнеры. Указанные ниже.

#1

Grafana+Prometheus

coinmarketcap-exporter, blackbox-exporter, grafana-image-renderer, node-exporter

Создал dashboard для мониторинга моего домашнего сервера на Raspberry Pi и настроил уведомления в Telegram, если загрузку RAM превысит 50%. Настроил Dashoards для мониторинга криптовалюты и их показателей c сервиса coinmarketcap.com, используя предоставляемые сервисом API.

#2

Gitea+Jenkins

maria db

Развернул контейнеры gitea и jenkins. Настроил Jenkins таким образом, что при commit в репозитория размещенного в Gitea, данные попадают на Jenkins конвейер и затем изменения попадаю на сервер сайта размещенный на DigitalOcean в директорию /vat/www/html/

#3

Nginx Proxy Manager

maria db

С его помощью настроил проксирование с внутренних IP своей машины на GCP, на dns записи созданные в GoDaddy. Таким образом оставив практически все порты закрытыми для внешнего мира.
